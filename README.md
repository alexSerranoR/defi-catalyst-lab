# DeFi Catalyst Lab

This repository is a personal learning and experimentation lab for a future TFG project related to DeFi, blockchain, graph analysis, and AI-assisted ecosystem research.

The goal of this repo is to collect notes, small experiments, API tests, prototypes, and early ideas before creating the final official TFG repository.

## Purpose

- Learn and test DeFi/blockchain data sources.
- Experiment with APIs such as DefiLlama, GitHub, and Snapshot.
- Build small Python prototypes.
- Explore graph analysis with NetworkX.
- Document useful notes and technical decisions.
- Keep early code and experiments separated from the final project repository.

## Status

This is not the final TFG repository.  
It is a sandbox for learning, testing, and prototyping.
